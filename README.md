# Data release for "Global time-frequency search for stellar-mass binary black holes in LISA"
[![DOI](https://zenodo.org/badge/1050035884.svg)](https://doi.org/10.5281/zenodo.17407705)  [![arXiv](https://img.shields.io/badge/arXiv-1234.56789-00ff00.svg)](https://arxiv.org/abs/1234.56789)

This repository contains the data products and scripts necessary to reproduce the Figures, Tables and conclusions presented in **Bandopadhyay, Chapman-Bird and Vecchio (2025)**

The results of search and follow-up parameter estimation are stored in `Shared_data_files`, along with the gapped dataset presented in Figure 1.
Timing information can be found in `Fig.3 and .4/timing-data`.
